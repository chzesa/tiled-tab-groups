Tab Catch

Tab Catch enables the use of rules to automatically assign tabs to groups based on the tab's url and/or title when navigating to a new URL.

There are two rule modes:

  • wildcards, e.g. '*.google.com' to match google.com and all of it's subdomains
  • regular expressions, e.g. 'Youtube\.com' for the expression '/Youtube\.com/i'
Use regular expressions matching mode

Rules created in wildcards mode also apply in regex mode, however any rules that have been last modified in regex mode are ignored if the current mode is wildcards mode.

If both 'Test Url' and 'Test Title' are unchecked the rule is ignored.
